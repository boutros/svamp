Application settings

These are the settings available to all of the application services, including middleware. Any changes you make here will take effect immediately in the running application server. So be carefull and don't touch anything you don't understand.

Additional, user-defined settings may be added to the file settings.edn by the system administrator.

The settings are automatically synced with the server when editing; there is no "save" button.
